Product overview

Bring WinGet into a friendlier desktop workflow

WinGet SuperTool is for people who want the convenience of software repository automation with a GUI that makes searching, bulk installs, and ongoing updates easier to manage.

Search and install software

Find packages and install them without memorizing WinGet syntax.

Update apps visually

Review and run updates in a more approachable interface for day-to-day maintenance.

Use template-based setups

Load application sets from CSV templates to speed up repeated machine prep.
