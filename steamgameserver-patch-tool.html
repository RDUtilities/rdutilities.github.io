Product overview

Automate the repetitive work around patching game servers

This tool is designed for server operators who want to reduce the manual steps around updates and keep scheduled maintenance more repeatable.

Start and stop servers

Manage the server lifecycle from a GUI rather than piecing commands together manually.

Patch on a schedule

Use Task Scheduler to automate maintenance windows and update routines.

Restart automatically

Bring the server back after patching so the full maintenance flow is more hands-off.
