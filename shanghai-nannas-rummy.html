Product overview

Two rummy experiences in one online game hub

Players can choose between Shanghai and Nanna's Rummy from the game status screen, making this a flexible web project that supports more than one card-play style.

Choose between two games

Switch between the predetermined meld structure of Shanghai and the more traditional rules of Nanna's Rummy.

Play solo or multiplayer

Use the web app for single-player sessions or create a lobby for multiplayer games.

Works on web and mobile

Designed to be reachable from desktop browsers and mobile devices for more flexible play.
